# 📝 To-Do List App

Aplicação de gerenciamento de tarefas desenvolvida com HTML, CSS e JavaScript puro.

## 🚀 Funcionalidades

* Adicionar tarefas
* Marcar como concluída
* Editar tarefas
* Deletar tarefas
* Filtro (todas, concluídas, pendentes)
* Persistência com localStorage

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript (Vanilla)
